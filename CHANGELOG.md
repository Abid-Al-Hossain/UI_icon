# Changelog

## 0.1.3 - 2026-03-31

- standardized icon preset metadata into explicit family, archetype, variant, and size fields
- upgraded preset filtering so the icon browser can slice the generated catalog by structured preset metadata instead of tags only
- split the old mixed icon basics panel into dedicated sizing and colors sections and normalized the editor spine toward a more button-class structure

## 0.1.2 - 2026-03-31

- expanded the preset system from starter packs into larger generated button-style preset catalogs
- aligned the preset browser and preview reset flow more closely with the button-component product model
- preserved clean lint, typecheck, and production build verification after the canon sync

## 0.1.1 - 2026-03-31

- added a searchable preset catalog with full-state preset apply flow
- aligned the icon studio more closely with the button-style UI Foundry editing model
- preserved clean lint, typecheck, and production build verification after the preset rollout

## 0.1.0 - 2026-03-30

- stabilized the standalone icon app so build, lint, and typecheck pass
- synced shared layout and control primitives to the UI Foundry standard
- preserved a self-contained icon playground and export workflow
