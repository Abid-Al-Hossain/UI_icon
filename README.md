# UI Foundry Icon Component Generator

Standalone visual icon generator built with Next.js.

This app is a focused icon-building studio for icon selection, container styling, transforms, effects, and animation that can be edited visually and exported as code.

## Product summary

This generator is designed for:

- designers building icon systems and action surfaces
- developers who want customizable icon presentation without rebuilding wrappers and effects manually
- freelancers and agencies preparing branded icon treatments for product UI
- buyers who want a standalone icon-styling tool that matches the rest of UI Foundry

## What the product includes

- section-based editing controls
- live preview panel
- code view with copy support
- file export support
- searchable preset library
- generated premium preset catalog with filters, paging, and surprise-me apply
- structured preset browsing with explicit family, archetype, and size filtering
- undo and redo controls
- self-contained standalone app structure

## Editing coverage

The editor currently includes these control areas:

- Presets
- Basics
- Library
- Sizing
- Colors
- Surface
- Transform
- Effects
- Motion
- Accessibility

These sections cover core rendering modes, icon source selection, stroke and fill systems, icon sizing, surface and container treatment, transforms, visual effects, motion, and accessibility metadata.

## Export workflow

The icon preview and code output reflect the same current state so users can design visually and export the matching implementation.

## Why this product is useful

This app is useful for teams that need more than raw icon glyphs. It turns icon treatment into a configurable product surface.
